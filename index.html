O que é o HTML?

(Voltar)

HyperText Markup Language, mais conhecido como HTML é uma linguagem de marcação criado pelo cientista da computação e físico Tim Berners-Lee no ano de 1991. Usado para estruturar conteúdos e dar corpo a uma página, o HTML é essencial para agrupar parágrafos e formar listas. Nele Podemos usar diferentes tipos de Tags para transformar e editar textos.

Estrutura do HTML

(Voltar)

Ao iniciar um projeto no HTML, é criado uma pasta onde o arquivo incial e os conteúdos visuais como fotos irão ficar. O arquivo criado se chama index.html e pode ser aberto atraves de um documento de texto e editado com o final .html para que possa funcionar no editor de código. São usadas as três tags primordiais para manter tudo em seu devido lugar, que são elas "html" que abre e fecha e que engloba as duas últimas, "head" para colocar o titulo da página e "body" onde fica todo o conteúdo da página a ser exibido.

O que são tags?

(Voltar)

As tags servem para estruturar e dar corpo a linguagem de marcação. Elas organizam e fazem com que o navegador interprete suas instruções. Para abrir e fechar uma tag usamos colchetes . Elas podem servir para deixar o texto em itálico com os elementos em “i” para abrir "/" pós colchete para indicar o fechamento da tag. É usado a tag “strong” para deixar o texto em negrito, entre outras.

Usamos atributos no HTML para criar um comportamento no elemento. Existem atributos genéricos, que podem ser utilizados em todas as tags, por exemplo: "id", "class" e "style". Existem atributos que são mais específicos, e só podem ser utilizados em certas tags. São eles "imput" que é usado para criar formulários juntamente com o atributo "type" para indicar o tipo de propriedade como por exemplo números ou seletor de cores.

Como utilizar listas

(Voltar)

A lista é um recurso muito importante no HTML, ela nos permite criar tópicos e ordernar textos. São muito usadas, e as mais conhecidas são as listas ordenadas e listas não ordenadas. As listas ordenadas são usadas para indicar um seguimento ou numeração. Utilizamos as tags "ol" e "li". A tag "o" é para definir o tipo da lista e a "li" para os itens. Nas listas não ordenadas não existe uma sequência, apenas os itens listados. Nessa utilizamos a tag “ul” e "li" igualmente para os itens.

Links no HTML

(Voltar)

Os links nos permite inserir endereços de destino sejam eles internos ou local. Podemos utilizar elementos como textos ou imagens. A tag referente ao link é "a" seguindo do atributo "href" que serve para redirecionar o usuário para um documento. Para abrirmos o documento em uma nova aba usamos o atributo "target" dentro da tag "a" junto com o atributo "_blank", "_self" para abrir na mesma janela, entre outros que são menos utilizados.
